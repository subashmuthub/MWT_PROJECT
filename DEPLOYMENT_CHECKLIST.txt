# Deployment Checklist

## Backend (Render.com)
- [ ] Create MySQL database
- [ ] Deploy web service from `zbackend`
- [ ] Link database to service
- [ ] Set FRONTEND_URL after Vercel deploy
- [ ] Test: https://your-backend.onrender.com/api/health

## Frontend (Vercel)
- [ ] Deploy from `frontend` folder
- [ ] Set VITE_API_URL env variable
- [ ] Test: https://your-app.vercel.app

## After Both Deployed
- [ ] Update FRONTEND_URL in Render backend
- [ ] Test login/register
- [ ] Test API calls in browser console
- [ ] Verify CORS working
- [ ] Initialize database (run migrations if needed)
